Proposed Morph Syntax

This page is a design sketch for named primitive morph declarations. It is not a runnable lattice_gen observable fixture yet, because the parser does not support this syntax today.

The idea is to use [...] for primitive element-by-element morph tables, and record-style {...} expressions for named structural product bridges:

Locality = [Global < Local] Regionality = [Global < Regional < Local] Uniqueness = [Aliased > Unique] Linearity = [Many < Once] Portability = [Portable < Shareable < Nonportable] Contention = [Contended > Shared > Uncontended] Forkable = [Forkable < Unforkable] Yielding = [Unyielding < Yielding] Staticity = [Dynamic > Static] Statefulness = [Stateless < Writing < Stateful; Stateless < Reading < Stateful] Visibility = [Immutable > Read > Read_write; Immutable > Write > Read_write] Externality = [External_ < External64 < Internal] Nullability = [Non_null < Maybe_null] Separability = [Non_float < Separable < Maybe_separable] Pointerness = [Non_pointer < Maybe_pointer] locality_as_regionality : Locality -> Regionality = [ Global -> Global; Local -> Local; ] regional_to_local : Regionality -> Locality = [ Global -> Global; Regional -> Local; Local -> Local; ] regional_to_global : Regionality -> Locality = [ Global -> Global; Regional -> Global; Local -> Local; ] linear_to_unique : Linearity -> Uniqueness^op = [ Many -> Aliased; Once -> Unique; ] unique_to_linear : Uniqueness^op -> Linearity = [ Aliased -> Many; Unique -> Once; ] portable_to_contended : Portability -> Contention^op = [ Portable -> Contended; Shareable -> Shared; Nonportable -> Uncontended; ] contended_to_portable : Contention^op -> Portability = [ Contended -> Portable; Shared -> Shareable; Uncontended -> Nonportable; ] statefulness_to_visibility : Statefulness -> Visibility^op = [ Stateless -> Immutable; Reading -> Read; Writing -> Write; Stateful -> Read_write; ] visibility_to_statefulness : Visibility^op -> Statefulness = [ Immutable -> Stateless; Read -> Reading; Write -> Writing; Read_write -> Stateful; ] Monadic = { uniqueness : Uniqueness; contention : Contention; visibility : Visibility; staticity : Staticity; } Comonadic = { linearity : Linearity; portability : Portability; forkable : Forkable; yielding : Yielding; statefulness : Statefulness; } Comonadic_with_locality = { areality : Locality; comonadic : Comonadic; } Comonadic_with_regionality = { areality : Regionality; comonadic : Comonadic; } Value = { comonadic : Comonadic_with_regionality; monadic : Monadic; } Alloc = { comonadic : Comonadic_with_locality; monadic : Monadic; } Scannable_axes = { externality : Externality; nullability : Nullability; separability : Separability; pointerness : Pointerness; } Kind_axes = { comonadic : Comonadic_with_locality; monadic : Monadic^op; scannable : Scannable_axes; } comonadic_locality_as_regionality : Comonadic_with_locality -> Comonadic_with_regionality = { areality = locality_as_regionality(areality); comonadic = comonadic; } comonadic_regional_to_local : Comonadic_with_regionality -> Comonadic_with_locality = { areality = regional_to_local(areality); comonadic = comonadic; } comonadic_regional_to_global : Comonadic_with_regionality -> Comonadic_with_locality = { areality = regional_to_global(areality); comonadic = comonadic; } comonadic_local_to_regional : Comonadic_with_locality -> Comonadic_with_regionality = { areality = locality_as_regionality(areality); comonadic = comonadic; } comonadic_to_monadic_min : Comonadic -> Monadic^op = { uniqueness = linear_to_unique(linearity); contention = portable_to_contended(portability); visibility = statefulness_to_visibility(statefulness); staticity = min; } comonadic_to_monadic_max : Comonadic -> Monadic^op = { uniqueness = linear_to_unique(linearity); contention = portable_to_contended(portability); visibility = statefulness_to_visibility(statefulness); staticity = max; } monadic_to_comonadic_min : Monadic^op -> Comonadic = { linearity = unique_to_linear(uniqueness); portability = contended_to_portable(contention); statefulness = visibility_to_statefulness(visibility); forkable = min; yielding = min; } monadic_to_comonadic_max : Monadic^op -> Comonadic = { linearity = unique_to_linear(uniqueness); portability = contended_to_portable(contention); statefulness = visibility_to_statefulness(visibility); forkable = max; yielding = max; } alloc_as_value : Alloc -> Value = { comonadic = comonadic_locality_as_regionality(comonadic); monadic = monadic; } alloc_to_value_l2r : Alloc -> Value = { comonadic = comonadic_local_to_regional(comonadic); monadic = monadic; } value_to_alloc_r2l : Value -> Alloc = { comonadic = comonadic_regional_to_local(comonadic); monadic = monadic; } value_to_alloc_r2g : Value -> Alloc = { comonadic = comonadic_regional_to_global(comonadic); monadic = monadic; }

In this sketch, users explicitly name the semantic product bridges the compiler cares about, while simpler structural morphs such as projections, min_with, and max_with can still be derived automatically.
